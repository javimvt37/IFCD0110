El formato de texto, es el marcado que se aplica a un texto para añadirle datos de forma. Al texto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido.

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios hojean las páginas por sus títulos. Es importante utilizar los encabezados para mostrar la estructura del documento.

Los encabezados <h1>deben ser utilizados para los encabezados principales, seguido por encabezamientos <h2>, <h3>, menos importantes y así sucesivamente.


Nota: Usar encabezados HTML para sólo encabezados. No utilice encabezados para que el texto GRANDE o negrita.

Etiquetas que dan formato al texto

HTML también define etiquetas especiales para la definición de texto con un especial significado .

HTML utiliza etiquetas para el formato de salida, como negrita o cursiva texto.

Etiqueta Descripción Ejemplo Utilización
<b> Define un texto en negrita Este texto está en negrita <b>Este texto está en negrita</b>
<strong> Define un texto importante, resaltado Este texto está en resaltado, es importante <strong>Este texto está en resaltado, es importante</strong>
<i> Define un texto en cursiva Este texto esta en cursiva <i>Este texto esta en cursiva</i>
<em> Define un texto enfatizado o con énfasis Este texto está enfatizado <em>Este texto está enfatizado</em>
<small> Define un texto más pequeño Este texto se muesta más pequeño <small>Este texto esta en small</small>
<sup> Define un texto en superíndice Este texto es superíndicede texto Este texto es <sup>superíndice</sup>
<sub> Define un texto en subíndice Este texto es subíndice de texto Este texto es <sub>subíndice</sub>
<ins> Define un texto insertado "añadido o agregado". Este texto es un texto insertado Este texto es un texto <ins>insertado</ins>
<s> Define un texto tachado Este texto es un texto tachado <s>Este texto es un texto tachado</s>
<u> Define un texto subrayadp Este texto esta subrayado <u>Este texto esta subrayado</u>
<del> Define un texto que ya ha sido eliminado o borrado Este texto está tachado eliminado Este texto está <del>tachado</del> eliminado
<mark> Define un texto marcado Este texto está marcado <mark>Este texto está marcado</mark>

Nota: Los navegadores muestran <strong> como <b>, y <em> como <i>. Sin embargo, hay una diferencia en el significado de las siguientes etiquetas: <b>y <i>define el texto en negrita y cursiva, pero <strong>y <em>significa que el texto es "importante".

Etiquetas que separan el texto

La etiqueta <p> separa el texto, añadiendo automáticamente un espacio (margen) antes y después de cada elemento <p>. El navegador eliminará todos los espacios en blanco y las líneas adicionales cuando se visualiza la etiqueta <p>.

La etiqueta <br> es una etiqueta vacía, lo que significa que no tiene etiqueta final. Inserta un salto de línea (una nueva línea) sin comenzar un nuevo párrafo.

Un texto dentro de una etiqueta <pre> se muestra en una fuente de ancho fijo (generalmente Courier), y conserva ambos espacios y saltos de línea.

Etiqueta Descripción Visualización Utilización
<p> Define un párrafo de texto

Esto es un párrafo de texto.

Esto es un párrafo de texto.

 <p>Esto es un párrafo de texto.</p> <p>Esto es un párrafo de texto.</p>
<br> Define un salto de línea. Esto es un salto de línea.

Esto es un salto de línea.
 Esto es un salto de línea.<br> <br> Esto es un salto de línea.<br>
<pre> Define un texto preformateado 
Esto es un texto preformateado.
Esto es un texto preformateado.
 <pre>Esto es un texto preformateado. Esto es un texto preformateado</pre>

Nota: Utilice el elemento <pre> cuando quiera mostrar el texto con formato inusual, o algún tiop de código informático.
