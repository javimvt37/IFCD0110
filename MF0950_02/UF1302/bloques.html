Etiquetas que citan los textos

Las abreviaturas de marcado <abbr> pueden propocionar información útil a los navegadores, sistemas de traducción y motores de búsqueda. Utiliza el atribute title para definir de la abreviación.

La etiqueta de marcado <cite> hace referencia a un trabajo creativo como un libro, una película, un programa de televisión o una canción, no el nombre de una persona.

La etiqueta de marcado <blockquote> define una cita larga que se representa como un elemento de bloque. No genera automáticamente comillas alrededor del contenido.

La etiqeuta de marcado <q> define una cita corta en línea. Marca las citas a otros autores o documentos. Genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <dfn> define un término y debe contener el atributo title para la definicion/explicación del término dentro.

Etiquetas de código de computadora

  • La etiqueta <code> define un fragmento de código de ordenador.
  • La etiqueta <kbd> representa comandos de netrada del usuario.
  • La etiqueta <samp> representa la salida de un programa o sistema informático.
  • La etiqueta <var> se utiliza cp,p expresión matemática o variable de la programación.

Etiquetas que separan el texto

La etiqueta <p> separa el texto, añadiendo automáticamente un espacio (margen) antes y después de cada elemento <p>. El navegador eliminará todos los espacios en blanco y las líneas adicionales cuando se visualiza la etiqueta <p>.

La etiqueta <br> es una etiqueta vacía, lo que significa que no tiene etiqueta final. Inserta un salto de línea (una nueva línea) sin comenzar un nuevo párrafo.

Un texto dentro de una etiqueta <pre> se muestra en una fuente de ancho fijo (generalmente Courier), y conserva ambos espacios y saltos de línea.
