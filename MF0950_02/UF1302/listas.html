Etiquetas que definen una tabla

Una tabla HTML se define con la <table> etiqueta.

Cada fila de la tabla se define con la <tr> etiqueta. Un encabezado de la tabla se define con la <th> etiqueta. Por defecto, encabezados de la tabla están en negrita y centrado. A los datos/celda de la tabla se define con la <td> etiqueta.


Etiqueta Descripción
<table> Define una tabla
<th> Define una celda de cabecera en una tabla
<tr> Define una fila en una tabla
<td> Define una celda en una tabla
<caption> Define el titulo o leyenda de una tabla
<colgroup> Especifica en una tabla un grupo de una o más columnas para darle formato
<col> Especifica propiedades de columna para cada columna dentro de un elemento <colgroup>
<thead> Agrupa el contenido de encabezado de una tabla
<tbody> Agrupa el contenido del cuerpo de una tabla
<tfoot> Agrupa el contenido del pie de página de una tabla

Diferentes tipos de listas

El atributo scope provee un mecanismo para indicar explícitamente a qué celdas afecta una celda de encabezado.El atributo scope especifica si una celda de encabezado es una cabecera de una columna, fila o grupo de columnas o filas. El atributo scope no tiene ningún efecto visual en los navegadores web normales, pero puede ser utilizado por los lectores de pantalla.
