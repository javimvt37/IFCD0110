HTML Vídeo

Antes de HTML5, un vídeo sólo se podía jugar en un navegador con un plug-in (como Flash). El HTML5 <video> elemento especifica una forma estándar para incrustar un vídeo en una página web.

HTML Object

El <object>elemento es soportado por todos los navegadores.

El <object>elemento define un objeto incrustado dentro de un documento HTML.
