¿Qué es CSS?

  • CSS significa Cascading Style Sheets (Hojas de Estilos en Cascada)
  • CSS describe cómo los elementos HTML se van a mostrar en la pantalla, papel, o en otros medios
  • CSS ahorra mucho trabajo . Se puede controlar el diseño de varias páginas web a la vez
  • Estilo externas se almacenan en archivos CSS

Ejemplo de las etiquetas básicas

  1. La etiqueta <!DOCTYPE html> define este documento para ser HTML5.
  2. La etiqueta <html> es el elemento raíz de una página HTML.
  3. El atributo lang="es-ES" especifica el idioma-país del documento y va dentro de la etiqueta <html>.
  4. La etiqueta <head> contiene información de metadatos sobre el documento.
  5. La etiqueta <meta charset="UTF-8"> define la codificación de caracteres.
  6. La etiqueta <title> especifica un título para el documento.
  7. La etiqueta <body> contiene el contenido de la página visible.
  8. La etiqueta <h1> define un gran encabezado.
  9. La etiqueta <p> define un párrafo.

Etiquetas HTML

Las etiquetas HTML son los nombres de elementos entre paréntesis angulares:

<Nombre de etiqueta> contenido va aquí ... </Nombre de etiqueta>

Etiquetas HTML
Normalmente vienen en pares como <p> y </p>.
Etiquetas HTML
Están formadas por una etiqueta de inicio y una etiqueta de fin.
La etiqueta final
Se escribe igual que la etiqueta de inicio, pero con una barra inclinada insertada antes del nombre de la etiqueta.

Consejo: La etiqueta de inicio también se llama etiqueta de apertura y la etiqueta final como etiqueta de cierre.

Estructura de una página HTML

<html>
<head>
<link rel="stylesheet" type="text/css" href="css/estilos.css">
<meta charset="UTF 8">
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>

Nota: Sólo el contenido dentro de la sección <body> (el área blanca arriba) se muestra en un navegador.
