Atributos

Los atributos porporcionan información adicional sobre los elementos HTML

HTML5 Attributes

HTML5 added the following attributes for <input>:

  • autocomplete
  • autofocus
  • form
  • formaction
  • formenctype
  • formmethod
  • formnovalidate
  • formtarget
  • height and width
  • list
  • min and max
  • multiple
  • pattern (regexp)
  • placeholder
  • required
  • step

and the following attributes for <form>:

  • autocomplete
  • novalidate

El Atributo maxlength

El atributo maxlength especifica la longitud máxima permitida para el campo de entrada:

Con un maxlengthatributo, el campo de entrada no aceptará más que el número de caracteres permitidos.

El maxlengthatributo no proporciona ninguna retroalimentación. Si desea alertar al usuario, debe escribir código JavaScript.
