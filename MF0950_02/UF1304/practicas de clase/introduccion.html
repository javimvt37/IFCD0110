El atributo del método

El methodatributo especifica el método HTTP ( GET o la POST ) para ser utilizado cuando la presentación de los datos del formulario:

<form action="/action_page.php" method="get">

<form action="/action_page.php" method="post">

Cuándo utilizar GET?

El método por defecto al presentar los datos del formulario es GET.

Sin embargo, cuando se utiliza GET, los datos de formularios presentados serán visibles en el campo de dirección de la página :

/action_page.php?firstname=Mickey&lastname=Mouse

Notas sobre GET:

  • Anexa forma-datos en el URL en el nombre de pares / valor
  • La longitud de un URL es limitado (aproximadamente 3000 caracteres)
  • Nunca utilice GET para enviar datos sensibles! (Serán visibles en la URL)
  • Útil para el envío de formularios donde un usuario desea marcar el resultado
  • GET es mejor para los datos no seguros, como las cadenas de consulta en Google

Atributos form

Aquí está la lista de todos los <form>atributos:

Atributo Descripción
aceptar-charset Especifica el conjunto de caracteres utilizado en el formulario enviado (por defecto: la página de juego de caracteres).
acción Especifica una dirección (URL) donde enviar el formulario (por defecto: la página de remitentes).
autocompletar Especifica si el navegador debe autocompletar la forma (por defecto: activado).
enctype Especifica la codificación de los datos presentados (por defecto: es codificada en URL).
método Especifica el método HTTP utilizado al presentar el formulario (por defecto: GET).
nombre Especifica un nombre usado para identificar la forma (para el uso de DOM: document.forms.name).
novalidate Especifica que el navegador no se debe validar el formulario.
objetivo Especifica el destino de la dirección en el atributo de acción (por defecto: _self).
