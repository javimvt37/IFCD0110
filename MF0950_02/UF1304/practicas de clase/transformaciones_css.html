Las transformaciones CSS

La transformpropiedad se aplica una transformación 2D o 3D a un elemento. Esta propiedad le permite rotar, escalar, mover, inclinación, etc., elementos.

El translate () Método

El translate()método se mueve un elemento desde su posición actual (de acuerdo con los parámetros dados para el eje X y el eje Y).

El ejemplo siguiente mueve el elemento <div> 50 píxeles a la derecha, y 100 píxeles hacia abajo desde su posición actual:

El rotate () Método

El rotate()método hace girar un sentido horario elemento o antihorario según un grado dado.

La escala () Método

Las scale()método aumenta o disminuye el tamaño de un elemento (de acuerdo con los parámetros dados para la anchura y altura).

El () Método skewX y skewY

El skewX()método sesga un elemento a lo largo del eje X por el ángulo dado.

El skewY()método sesga un elemento a lo largo del eje Y por el ángulo dado.

La matriz () Método

El matrix()método combina todo el 2D métodos de transformada en una sola.

El método de la matriz () tomar seis parámetros, que contiene funciones matemáticas, que le permite rotar, escala, movimiento (traducir), y el sesgado elementos.

Los parámetros son los siguientes: matriz (scaleX (), skewY (), skewX (), scaleY (), translateX (), translateY ())
